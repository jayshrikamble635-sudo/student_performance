# Student Performance Predictor

## Overview

This project is a simple web application built using Streamlit.
It predicts student performance based on study hours, attendance, and sleep hours.

## Features

* User-friendly interface
* Takes input from user
* Displays predicted performance score
* Built with Python and Streamlit

## Requirements

Install the required libraries using:

```
pip install streamlit
```

## How to Run

1. Save the file as `app.py`
2. Open terminal in project folder
3. Run the command:

```
streamlit run app.py
```

4. The app will open in your browser

## Project Structure

```
project-folder/
│
├── app.py
├── README.md
```

## Input Parameters

* Hours Studied
* Attendance Percentage
* Sleep Hours

## Output

The app calculates and displays the predicted performance score.
